# Est-ce qu'il pleut à Nantes ?

Une page web minimaliste qui affiche le nombre de jours consécutifs sans pluie à Nantes.

## Fonctionnalités

- Affichage en temps réel du nombre de jours sans pluie
- Données météo provenant de l'API [Open-Meteo](https://open-meteo.com/) (gratuite, sans clé API)
- Design adaptatif (mobile et desktop)
- Interface entièrement en français

## Démo

La page est accessible à l'adresse : [https://nombredejoursanspluie.lhomme.xyz](https://nombredejoursanspluie.lhomme.xyz)

## Fonctionnement technique

- Récupération des données de précipitations des 30 derniers jours via l'API Open-Meteo
- Calcul du nombre de jours consécutifs sans pluie en partant d'aujourd'hui
- Changement de couleur de fond selon qu'il pleut ou non
